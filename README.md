# Warehouse Inventory Management System

## Description
This is a Java-based console application that manages warehouse inventory using Java Collections Framework.

 Features
- Add items
- Remove items
- Update quantity
- Search by ID and Name
- Display all items

 Technologies Used
- Java
- HashMap (Collections Framework)

 How to Run
1. Compile all files:
   javac *.java

2. Run:
   java Warehouse

Sample Output
=== Warehouse Menu ===
1. Add Item
2. Remove Item
3. Update Quantity
4. Search by ID
5. Search by Name
6. Display All Items
7. Exit
